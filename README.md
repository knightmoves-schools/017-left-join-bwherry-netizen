The SQL in the `exercise.sql` file
- Should return a table that shows each employees `NAME` titled as "Employee Name", next to their `ROLE_NAME` titled as "Job Title"
- Should associate Employee's name and role info based on their respective IDs, and display all `ROLE_NAME`s in the `ROLES` table even if there is no employee associated with that role
